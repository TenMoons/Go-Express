# 简介

「拜托了快递」 是 **AHU校园快递代取平台**，是AHU软件工程综合实践作品。基于微信小程序原生语法实现，使用

[Lin UI]: https://github.com/TaleLin/lin-ui

和自定义组件，采用七牛云对象存储；后端

[后端]: https://github.com/TenMoons/go-express-django-backend

使用Django实现。
